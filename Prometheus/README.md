# Prometheus Monitoring Workshop

## Overview

A 60-minute hands-on workshop on Prometheus. Starts with monitoring fundamentals, covers Prometheus architecture, runs Prometheus + Alertmanager with Docker Compose, writes a basic alert rule, and routes alerts to Discord.

## Target Audience

- Comfortable with Docker and Docker Compose (completed the Docker workshop)
- Interested in observability, monitoring, or SRE work
- No prior Prometheus experience required
